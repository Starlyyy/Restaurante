**Sistema Restaurante OverCozido**
- Trabalho elaborado por Izabella da Costa Codo e Rebeca Schmidt - 3° TDS
- Com ajax/js foram feitas as seguintes implementações:
    - Validação de campos;
    - Inserção no banco de dados;
    - Carregamento de dados no select.
 
- Estilo das páginas foi feito com Bootstrap e CSS.
